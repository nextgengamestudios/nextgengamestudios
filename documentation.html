Instalacja

Aby zainstalować najnowszą wersję SDK (1.0.0_beta_4), twój komputer musi spełniać poniższe wymagania sprzętowe:
- Procesor: 2 rdzenie (1.7GHz per core)
- Pamięć RAM: 8GB
- Karta graficzna: kompatybilna z DX11
- Miejsce na dysku: 3GB
- System operacyjny: Windows 8.1, Windows 10, Windows 11

Jeżeli twój komputer jest kompatybliny, zainstaluj SDK używając jednej z tych komend (zależnie od używanej konsoli, wymagają uprzedniej instalacji gry)
- CMD: > nextgen install sdk-latest
- PowerShell: > nextgen shell --install -p sdk-latest
